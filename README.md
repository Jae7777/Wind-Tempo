# Wind-Tempo

**Wind Tempo** is a rhythm-based piano game inspired by popular rhythm titles like *OSU!* and *Guitar Hero*.  
Instead of tapping circles or strumming guitars, players use a **MIDI/Digital Keyboard** to hit falling notes in sync with the music.  

The goal is to create a game thatâ€™s fun and educational, helping players improve rhythm, timing, and piano skills.

---

## Planned Features

- Falling-note gameplay.  
- Support for MIDI and digital keyboards.  
- Multiple difficulty levels (Easy / Medium / Hard).  
- Real-time scoring system based on timing accuracy.  
- Visual feedback (Perfect / Great / Good / Miss).  
- Expandable song library with custom charts.  

## Project Goals

- Build a playable beta with a basic note highway.  
- Implement real-time input detection for MIDI/Digital keyboards.  
- Create a scoring and combo system.  
- Add UI elements (score, combo, accuracy).  
- Add visuals, menus, and multiple songs.
