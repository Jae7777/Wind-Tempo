# Wind-Tempo

## Overview
**Wind Tempo** is a rhythm-based piano game inspired by popular rhythm titles like *OSU!* and *Guitar Hero*.  
Instead of tapping circles or strumming guitars, players use a **MIDI keyboard** or a **digital keyboard** to hit falling notes in sync with the music.  

The goal of this project is to create a game thatâ€™s both **fun and educational**, helping players improve rhythm, timing, and piano skills.

---

##  Planned Features
- Falling-note gameplay.  
- Support for MIDI and digital keyboards.  
- Multiple difficulty levels (Easy / Medium / Hard).  
- Real-time scoring system based on timing accuracy.  
- Visual and audio feedback (Perfect / Great / Good / Miss).  
- Expandable song library with custom charts.

##  Project Goals
1. Build a working beta with a basic note highway.  
2. Implement real-time input detection (MIDI + digital).  
3. Create a scoring and combo system.  
4. Add UI elements (score, combo, accuracy).  
5. Add visuals, menus, and multiple songs.
