# Wind-Tempo

## Overview
**Wind-Tempo** is a rhythm-based piano game inspired by popular rhythm titles such as *OSU!* and *Guitar Hero*.
Instead of tapping circles or strumming guitars, players use a **Digital/MIDI Keyboard** to hit falling notes in sync with the music.

---

## Planned Features
- Falling-note gameplay
- Falling-note "note highway" gameplay.  
- Support for MIDI/Digital keyboards.  
- Multiple difficulty levels (Easy / Medium / Hard).  
- Real-time scoring system based on timing accuracy.  
- Visual feedback (Perfect / Great / Good / Miss).  
- Expandable song library with custom charts. 

---

## Project Goals
- Build a playable beta with a basic note highway
- Implement real-time input detection for Digital/MIDI keyboards.
- Create a scoring and combo system
- Add UI elements (score, combo, accuracy) 
- Add visuals, menus, and multiple songs.
